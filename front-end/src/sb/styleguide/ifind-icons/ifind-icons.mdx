# IFIND Icons

## Usage

### Copy Icons

The SVG assets needs to be copied into the public directory of the project.
This can usually be done by adding a copier script when the server starts.

```js
// load-icons.js

const path = require("path");
const ifindIconsPath = require.resolve("ifind-icons");
const copyIconsTo = require(path.resolve(ifindIconsPath, "../copy-icons-to"));

copyIconsTo(path.resolve("./public"));
```

### Component Usage

Sample:

```js
import IfindIcon from '@componens/IfindIcon';

const SomeComponent = () ={
  return (
    <div>
      <IfindIcon icon="play" />
    </div>
  )
}
```

### Dark Variant

```js
<IfindIcon icon="play" dark />
```

### Styling

Supply a className prop to the IfindIcon in order to override styles in scss.

```js
// SomeComponent.js
<IfindIcon icon="play" dark className="my-icon" />
```

```css
/* .scss */
.my-icon {
  color: color(red);
}
```
